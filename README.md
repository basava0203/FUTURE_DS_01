A dynamic Power BI dashboard built to analyze and compare sales performance across two key fiscal years: 2009-2010 and 2010-2011.

Overview
This project provides an interactive visualization of sales data, enabling stakeholders to track key metrics, identify trends, and perform year-over-year comparisons. The dashboard is designed to offer actionable insights at a glance.

Key Features
Year-over-Year Comparison: Easily toggle between the 2009-2010 and 2010-2011 periods to assess growth and trends.

Interactive Visualizations: Includes a variety of charts, graphs, and KPIs to drill down into the data.

Sales & Performance Metrics: Tracks critical metrics such as Total Sales, Profit, and Quantity Sold.
